# `v810-unknown-vb`

**Tier: 2**

The `v810-unknown-vb` target is a compilation target for the Virtual Boy. It has no std support, but full support for the `core` crate.

## Target maintainers

[@SupernaviX](https://github.com/SupernaviX)

## Requirements

This target is for cross-compilation only.

## Building the target

Download prebuilt binaries from [https://github.com/SupernaviX/v810-rust/releases].

## Building Rust programs

Check out [https://github.com/SupernaviX/rust-vb-platform].
